## Run Locally

Clone the project

```bash
  git clone https://github.com/devTarak/Ultimate-Java-Mastery-Course
```

Go to the project directory

```bash
  cd Ultimate-Java-Mastery-Course
```

Run java File

```bash
 javac filename.java
 java filename
```
